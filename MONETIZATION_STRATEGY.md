### 4. Analytics Loop (Optimize While You Sleep)

**Key Metrics to Track:**

| Metric | Target | What to Optimize |
|--------|--------|------------------|
| Chat ‚Üí Qualification | 60%+ | Chat UX, AI prompting |
| Qualification ‚Üí Email Capture | 40%+ | Value prop, email modal copy |
| Results ‚Üí Click | 30%+ | Program presentation, trust signals |
| Click ‚Üí Application Start | 50%+ | Affiliate link quality, partner UX |
| Application ‚Üí Complete | 20%+ | Partner's funnel (can't control, but matters) |

**Weekly Review (15 minutes):**
```sql
-- Run this query every Monday
SELECT
  date_trunc('week', created_at) as week,
  COUNT(*) as total_clicks,
  COUNT(DISTINCT session_id) as unique_users,
  COUNT(*) FILTER (WHERE application_started) as applications,
  ROUND(100.0 * COUNT(*) FILTER (WHERE application_started) / COUNT(*), 2) as conversion_rate
FROM program_clicks
WHERE created_at >= CURRENT_DATE - INTERVAL '8 weeks'
GROUP BY 1
ORDER BY 1 DESC;
```

**What to Do With Data:**
- Conversion rate dropping? A/B test new CTAs
- Certain programs converting better? Feature them more prominently
- Email open rates low? Test new subject lines
- High traffic but low conversions? Add more trust signals

**Automation:**
Set up Supabase scheduled function to email you this report every Monday at 9am.
